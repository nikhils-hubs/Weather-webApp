# 🌦️ Weather App

A clean and responsive Weather App built using **HTML**, **CSS**, and **JavaScript** that fetches real-time weather data using a Weather API.  
Users can search any city and instantly get current weather details like temperature, humidity, wind speed, and weather conditions.

---

## 🚀 Features

- 🌍 Search weather by city name
- 🌡️ Real-time temperature display
- 💨 Wind speed information
- 💧 Humidity details
- ☁️ Dynamic weather conditions/icons
- 📱 Responsive UI for mobile and desktop
- ⚡ Fast and lightweight

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Weather API
